# Backup Scripts

Scripts utilitaires pour créer des archives du projet.

## Fichiers

### create-claude-archive.sh
Crée une archive complète du projet pour Claude Code.

### create_claude_code_archive.sh
Crée une archive code-only (sans logs/backups) pour Claude Code.

## Usage

```bash
# Archive complète
./scripts/backup/create-claude-archive.sh

# Archive code uniquement
./scripts/backup/create_claude_code_archive.sh
```
