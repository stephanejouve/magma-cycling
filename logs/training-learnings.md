## Enseignements Clés Établis

### ✅ Capacité Terrain >> Indoor
- **Découverte** : 54km outdoor RPE 4/10 = 163 TSS (surpasse largement capacité indoor équivalente)
- **Implication** : Tolérance volume/intensité augmentée en extérieur
- **Problématique** : Discipline intensité défaillante → surcharge systématique
- **Solution** : Stratégie indoor-only 2-3 mois pour établir discipline

### ✅ Sommeil = Facteur Limitant
- **Observation** : Qualité sommeil détermine directement exécution VO2
- **Données** : Moyenne 5h33 vs cible 7h+
- **Impact validé** : Échec VO2 si <7h sommeil veille
- **Action** : Sommeil comme critère strict checklist VO2

### ✅ Sweet-Spot Consolidé
- **Intensité validée** : 88-90% FTP
- **Format optimal** : 3x8min avec récup 3min à 65% FTP
- **Progression** : 88%+ à évaluer selon TSB et validation

### ✅ Découplage Cardiovasculaire <7.5%
- **Seuil établi** : <7.5% = validation qualité entraînement
- **Observation** : Systématiquement respecté sur séances qualité
- **Usage** : Indicateur post-séance fiable

### ✅ Asymétrie Pédalage Variable
- **Pattern** : Effort-dépendant
  - Explosif (>110% FTP) : 56% droite / 44% gauche
  - Modéré (75-95% FTP) : 50% / 50% (équilibre)
- **Interprétation** : Physiologique, pas pathologique
- **Action** : Surveillance continue, pas d'intervention

## Protocoles Validés

### Checklist VO2 Max (5 Critères Obligatoires)
1. ✅ TSB ≥ +5
2. ✅ Sommeil ≥ 7h la veille
3. ✅ Aucune intensité >85% FTP dans les 48h précédentes
4. ✅ FC repos matinale normale (40 ± 5 bpm)
5. ✅ Aucun signe fatigue résiduelle

**Validation** : 250W (114% FTP220) testée avec succès

### Hydratation Intensité >88% FTP
- **Protocole** : Fréquence doublée des prises
- **Format** : Isotonique
- **Validation** : Impact positif observé

### Nutrition Terrain
- **Capacité individuelle** : Maximum 45g glucides/heure
- **Timing optimal** : 15min avant efforts principaux (montées >5%)
- **Format privilégié** : Rice cakes (tolérance validée)
- **Planification** : Waypoints automatiques RideWithGPS → Wahoo
