# Enseignements d'Entraînement S069

## 1. Dissociation Performance Ponctuelle vs État Systémique Réel

### Découverte Majeure
Maintien de métriques techniques exceptionnelles (découplages 0.2%, 0.6%, 2.0%, 4.9%) en dette sommeil chronique (3 nuits <6h) ne reflète pas capacité récupération réelle. VFC révèle surcharge système nerveux autonome (81ms→51ms, -37% en 48h) invisible dans performance immédiate.

### Pattern Identifié
- **Performance préservée court terme** : Découplage, puissance, cadence maintenus en Z2
- **Récupération compromise moyen terme** : VFC chute prédictive symptômes +24h
- **Limitation progression** : Capacité haute intensité (sweet-spot >88% FTP) bloquée même si Z2 exécutable

### Implications Protocole
1. Monitoring VFC quotidien obligatoire âge 54, sensibilité surmenage accrue
2. Performance séance isolée ≠ validation état général
3. Seuils décisionnels VFC établis :
   - <50ms : Repos complet
   - 50-55ms : Z1 uniquement (VETO intensité)
   - 55-60ms : Z2 possible, surveillance
   - >60ms : Progression intensité modérée autorisée
   - >70ms : Intensité haute (VO2) possible si sommeil 7h+

### Application S070
VFC monitoring matin avant décision séance quotidienne. Pas de sweet-spot avant stabilisation >60ms + sommeil 6h30+ consolidé 3 nuits consécutives.

## 2. Sommeil = Facteur Limitant Progression Intensité

### Quantification Impact
Dette sommeil 3 nuits <6h (4.8h-5.4h-6.5h) compatible exécution Z2 (IF 0.63-0.73, découplages bons) mais empêche progression sweet-spot authentique (88-90% FTP). Séance étiquetée "sweet-spot" (S069-02) réellement tempo modéré (IF 0.73 = 73% FTP vs 88-90% attendu).

### Seuils Identifiés
- **5-6h sommeil** : Z2 exécutable, qualité technique préservée
- **<5h sommeil** : Risque VFC <55ms, VETO intensité >75% FTP
- **6h30-7h sommeil** : Minimum requis sweet-spot (88% FTP)
- **7h+ sommeil** : Requis VO2 max (>106% FTP)

### Validation Données
- S069-01 : Sommeil 5.7h, VFC 81ms → Z2 excellent (découplage 0.2%)
- S069-02 : Sommeil 4.8h → Tempo seulement (IF 0.73), pas sweet-spot vrai
- S069-03 : Sommeil 5.4h, VFC 51ms → Z1 technique acceptable, intensité impossible
- S069-04 : Sommeil 6.5h, VFC 61ms → Z2-force OK, récupération validée

### Protocole Établi
**Matrice Décision Sommeil x VFC :**
| Sommeil | VFC >60ms | VFC 50-60ms | VFC <50ms |
|---------|-----------|-------------|-----------|
| >7h | VO2 possible | Sweet-spot OK | Z2 max |
| 6-7h | Sweet-spot OK | Z2 max | Z1 max |
| 5-6h | Z2 max | Z1 max | Repos |
| <5h | Z1 max | Repos | Repos |

## 3. VFC = Indicateur Prédictif Fiable Vulnérabilité

### Pattern Validation
Chute VFC 51ms mercredi (J3) précède apparition symptômes toux 24-36h plus tard (jeudi soir/vendredi). Corrélation établie : VFC <55ms = fenêtre vulnérabilité immunitaire accrue.

### Timeline Détaillée
- **Lundi (J1)** : VFC 81ms, sommeil 5.7h → État optimal
- **Mardi (J2)** : VFC 60ms* (estimé), sommeil 4.8h → Début dégradation
- **Mercredi (J3)** : VFC 51ms, sommeil 5.4h → Alerte critique émise
- **Jeudi (J4)** : VFC 61ms (récupération), toux apparition soir → Symptômes déclarés
- **Vendredi (J5)** : Arrêt entraînement, confirmation rhinopharyngite

*Données fragmentaires

### Fenêtre Prédictive
VFC <55ms donne avertissement 24-48h avant manifestation symptômes cliniques. Permet intervention préventive (repos, hydratation, sommeil prioritaire) potentiellement atténuante.

### Validation Rétrospective
Si VETO intensité mercredi (J3) avait inclus repos complet jeudi (vs séance force outdoor réalisée), évolution maladie possiblement différente. Séance force Prospect Park (TSS 53, RPE 6/10) a pu accélérer basculement vers symptômes.

### Seuil Critique Confirmé
VFC <55ms = seuil alerte rouge, repos ou Z1 pure (<50% FTP) obligatoire. Pas d'exceptions, même si découplage/performance ponctuels corrects.

## 4. Stratégie Indoor-Only Haute Intensité Confirmée

### Validation Prospect Park (S069-04)
Sortie outdoor force-endurance Prospect Park acceptable en récupération (TSS 53, IF 0.67, découplage 4.9%) mais écarts significatifs vs consignes indoor structurées :
- Cadence 82rpm moyenne vs 85-88rpm cible (blocs récupération sous-exécutés)
- Relief 6 montées impose variations cadence/puissance non prévues
- RPE 6/10 élevé pour Z2 cohérent avec force 70rpm + dénivelé

### Rappel Historique
Capacité terrain >> indoor (54km outdoor RPE 4/10 = 163 TSS surpasse largement indoor) mais discipline intensité outdoor défaillante historique. Surcharges systématiques terrain ont justifié stratégie indoor-only actuelle.

### Décision Maintenue
Séances intensité >85% FTP restent indoor-only jusqu'à validation discipline (2-3 mois minimum). Outdoor acceptable pour :
- Récupération Z1-Z2 (IF <0.70)
- Exploration technique force-endurance légère
- Contexte social/motivation (Foudre validé)

Sweet-spot (88-90% FTP), seuil (95-105% FTP), VO2 (106-120% FTP) exclusivement indoor.

## 5. Arrêt Préventif = Décision Économiquement Rationnelle

### Analyse Coût/Bénéfice
**Scénario A (Arrêt immédiat J5)** :
- CTL érodé : -4.6 points (-8%)
- TSB final : +19 (surcondition temporaire)
- Durée arrêt estimée : 5-7 jours
- Reprise progressive : S070 Z2, S071 sweet-spot possible

**Scénario B (Hypothétique poursuite entraînement malade)** :
- Risque aggravation rhinopharyngite → bronchite
- CTL érosion potentielle : -10 points minimum (2-3 semaines arrêt)
- Reprise retardée : S072-S073
- Perte nette supplémentaire : 2-3 semaines progression

### ROI Décision
Arrêt préventif préserve :
- Santé long terme (pas complications pulmonaires)
- Capital fitness (CTL 52.5 vs <47 scénario B)
- Timeline progression (retour S070 vs S072-S073)

### Validation Protocole
Toux + VFC instable semaine + dette sommeil = triggers arrêt non négociables âge 54. Pas de "push through", récupération santé prioritaire sur TSS hebdomadaire.

### Application Future
Premiers symptômes (toux, gorge) + VFC <55ms = repos immédiat 48-72h. Test reprise 35min Z1 (50-60% FTP) validation avant progression. Pas compensation volume post-guérison, reconstruction progressive CTL.
