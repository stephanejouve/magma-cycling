# Adaptations de Protocoles S068

## 1. Sweet-Spot : Cadence Optimale Établie

### Ajustement Seuil
**Ancien protocole** : Cadence Sweet-Spot 85-92rpm  
**Nouveau protocole** : Cadence Sweet-Spot **90-95rpm** (zone optimale validée)

### Justification Data
- S068-02 : Cadence 85rpm → découplage 2.7% (bon mais sous-optimal)
- S068-04 : Cadence 90-91rpm → découplage blocs 3.1%-4.8%-5.2% (excellent) + équilibre G/D 50/50

### Critères Validation
Cadence 90-95rpm améliore :
1. Efficacité neuromusculaire (équilibre G/D)
2. Qualité aérobie (découplage réduit)
3. Confort exécution (RPE cohérent IF)

### Application S069
- **Template Sweet-Spot** : Cadence 90-95rpm obligatoire sur blocs intensité
- **Échauffement/Récup** : Cadence libre 85-90rpm tolérée
- **Monitoring** : Vérifier cadence moyenne intervalles ≥90rpm systématiquement

## 2. Drills Unijambiste : Intensité Réduite

### Ajustement Critère
**Ancien protocole** : Drills unijambiste 75% FTP × 90s  
**Nouveau protocole** : Drills unijambiste **60-65% FTP × 45-60s**

### Justification Data
S068-05 : Drills incomplets car puissance cible 75% FTP trop élevée pour maintenir 90s en unijambiste. FC pic 126bpm suggère intensité excessive pour exercice technique.

### Objectifs Clarifiés
Drills unijambiste = **travail neuromusculaire**, pas test endurance. Priorités :
1. Maintien cadence 80-85rpm fluide
2. Équilibre plateau pédale sans compensation bassin
3. Coordination mouvement circulaire

Puissance secondaire → réduire intensité pour préserver qualité technique.

### Application S069
- **Format** : 3-4 sets × (45s jambe G + 45s jambe D + 90s normal)
- **Intensité** : 60-65% FTP
- **Placement** : Post-échauffement ou cooldown, jamais sur efforts principaux
- **Validation** : Cadence maintenue >80rpm sur toute durée

## 3. Badges/Côtes : Restriction Stricte

### Nouveau Protocole Discipline
**Application immédiate S069** :

1. **INTERDICTION absolue** ajouts badges/côtes/sprints spontanés en cours séance (hors jours récup dédiés)
2. **Si badge non terminé** : Anticiper +15min durée séance dès planification OU reporter jour récup
3. **Jours récup dédiés** : Badges autorisés UNIQUEMENT sur séances REC prévues RPE max 3/10, durée flexible

### Justification Data
S068-05 : Surcharge +33% TSS (+15 points) suite ajouts non planifiés badge + côte + sprint. Pattern reproduit historique S065-S067 terrain.

### Critères Exclusion
- Badges complexes (>20min ou >50 TSS estimé) = séance dédiée uniquement
- Côtes simples (<3min) = tolérées sur jours récup uniquement
- Sprints 5-10s = INTERDITS hors séances neuromusculaires prévues

### Monitoring S069-S071
Tracker respect protocole 3 semaines consécutives. Si validation (0 écart), envisager retour progressif terrain S072+.

## 4. Sommeil : Seuils VETO Formalisés

### Critères VETO Séances Intensité

**VETO VO2 max (106-120% FTP)** :
- Sommeil <6h ET/OU score Withings <60 ET/OU VFC <35ms
- Report séance ou substitution Z2 endurance

**VETO Sweet-Spot haute intensité (>90% FTP)** :
- Sommeil <6h ET VFC <40ms ET/OU score Withings <60
- Réduction intensité à 88% FTP OU report 24h

**Tolérance Sweet-Spot standard (88-89% FTP)** :
- Sommeil >5h ET VFC >35ms ET score Withings >50
- Exécution autorisée avec surveillance RPE (max 6/10)

### Justification Data
S068-02 : Sommeil 47 score + VFC 32ms → séance Sweet-Spot 88% exécutée avec qualité (découplage 2.7%) mais RPE 5/10 élevé pour intensité. Contexte limite acceptable uniquement en phase consolidation, inadapté pour progression intensités >90%.

### Application S069
1. **Monitoring systématique** : Score sommeil + VFC au réveil avant chaque séance intensité
2. **Décision data-driven** : Appliquer critères VETO sans exception
3. **Substitutions prévues** : Templates Z2 alternatifs pré-générés si VETO activé

## 5. Hydratation : Fréquence Séances Sweet-Spot

### Ajustement Protocole
**Ancien protocole** : Hydratation standard 500ml/h toutes séances  
**Nouveau protocole** : Hydratation **renforcée 600ml/h** séances Sweet-Spot >60min

### Justification Physiologique
Sweet-Spot 88-90% FTP = intensité seuil transition aérobie/anaérobie. Production chaleur et sudation augmentées vs Z2. S068-04 : 70min Sweet-Spot sans déshydratation apparente (découplage maîtrisé) mais protocole préventif recommandé.

### Critères Application
- **Sweet-Spot <60min** : 500ml/h standard
- **Sweet-Spot 60-90min** : 600ml/h (150ml/15min)
- **Sweet-Spot >90min** : 700ml/h + isotonique

### Application S069
Prévoir bidons supplémentaires séances Sweet-Spot prévues S069-02 (si 3×10min) et S069-04.

## 6. TSS Hebdomadaire : Cible Augmentée

### Ajustement Cible
**Ancien protocole** : TSS hebdomadaire 350-380 (zone maintenance)  
**Nouveau protocole** : TSS hebdomadaire **400-420** (zone progression)

### Justification Objectif
CTL progression S068 : +0.3 insuffisant pour atteindre FTP 260W en 20 semaines (nécessite +0.85-1.1 CTL/semaine).

### Répartition Recommandée S069
- **Endurance Z2** : 4 séances × 60-90min = ~240-280 TSS (60-65%)
- **Sweet-Spot 88-92%** : 2 séances × 60-75min = ~120-140 TSS (30-33%)
- **Récupération** : 1 séance × 40-50min = ~25-30 TSS (6-7%)
- **Total** : 385-450 TSS (cible 400-420)

### Monitoring S069-S071
1. **Semaine 1 (S069)** : Viser 400 TSS (test capacité absorption)
2. **Semaine 2 (S070)** : Ajuster 390-420 selon TSB évolution
3. **Semaine 3 (S071)** : Consolider 400-420 si validation CTL progression >+0.8

## Exclusions Maintenues

### Formats Interdits (Confirmés S068)
1. **Over/Under >85% baseline** : Trop exigeant récupération cardiovasculaire actuelle (VFC faible)
2. **Sorties terrain non supervisées** : Pattern surcharge validé, indoor-only maintenu S069-S071 minimum
3. **Séances >2h** : Capacité absorption limitée dette sommeil chronique
4. **HIIT pur (>120% FTP)** : Introduction différée post-validation Sweet-Spot 92% FTP

### Surveillance Renforcée S069

1. **Respect protocole discipline** : 0 ajout spontané badges/côtes
2. **VFC quotidienne** : Seuils VETO appliqués systématiquement
3. **TSS hebdomadaire** : Tracker progression 400-420 sans dépassement >10%
4. **Équilibre G/D** : Mesurer sur toutes séances intensité (Sweet-Spot + VO2 si introduit)
