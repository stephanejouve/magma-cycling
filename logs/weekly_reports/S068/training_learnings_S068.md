# Enseignements d'Entraînement S068

## 1. Validation Progression Sweet-Spot 88-90% FTP

### Découverte Technique
Deux séances Sweet-Spot (S068-02 et S068-04) confirment capacité maintenir intensité 88-90% FTP avec qualité aérobie excellente :
- **S068-02** : 2×10min @ 192-193W (88% FTP), découplage 2.7%
- **S068-04** : 3 blocs @ 196-197W (89-90% FTP, 28min cumulés), découplage max 5.2%

### Preuves Objectives
- Découplage tous <7.5% (seuil validation qualité)
- Stabilité puissance remarquable : ±1-2W variation entre intervalles
- Cadence 90-91rpm optimale validée sur S068-04 (vs 85rpm sous-optimal S068-02)

### Protocole Validé
- **Échauffement** : 12-15min ramp 50-65% + 3-5min @ 65% cadence libre
- **Blocs Sweet-Spot** : 88-90% FTP, durée 8-10min, cadence 90-95rpm
- **Récupération inter-blocs** : 2-3min @ 60-65% FTP
- **Prérequis** : TSB ≥-2, sommeil score ≥60, VFC >40ms

### Progression Recommandée
- **S069** : Tester 3×10min @ 90% FTP OU 2×12min @ 89% FTP (selon TSB/sommeil)
- **S070-S071** : Introduction progressive 92% FTP sur volumes réduits (2×8min)

## 2. Équilibre Pédalage G/D Dépendant de l'Intensité

### Pattern Identifié
- **Sweet-Spot (88-90% FTP)** : Équilibre excellent 50/50 ou 49/51
- **Explosif (>120% FTP)** : Asymétrie connue 56/44 droite/gauche

### Explication Physiologique
Sweet-Spot sollicite fibres aérobies (Type I) uniformément distribuées, tandis que efforts explosifs recrutent fibres anaérobies (Type II) avec déséquilibre latéralisation neuromusculaire.

### Application Pratique
Sweet-Spot = **format optimal correction déséquilibres** long terme. Intégrer systématiquement dans cycles base (Cycle 1-2 périodisation) pour rééquilibrage progressif.

### Surveillance S069
Mesurer équilibre G/D sur séances VO2 max (106-120% FTP) si introduites. Hypothèse : asymétrie augmente avec intensité anaérobie.

## 3. Discipline Planning Indoor : Pattern Surcharge Reproduit

### Observation S068-05
Séance endurance technique Z2 : TSS réalisé 61 vs 46 prévu (+33%), durée 76min vs 62min (+14min).

**Causes identifiées** :
- Ajout badge circuit non anticipé
- Côte supplémentaire non planifiée
- Sprint 5s spontané

### Comparaison Historique
Pattern identique observé S065-S067 terrain : IF dépassements systématiques +13% à +38% malgré monitoring outils et conscience athlète.

### Protocole Invalidé
Ajouts spontanés badges/côtes/sprints en cours séance = **compromis contrôle TSS hebdomadaire** et risque surcharge non planifiée.

### Protocole Établi S069
1. **Si badge non terminé** : Anticiper +15min durée séance dès planification
2. **Jours récupération dédiés** : Badges/côtes autorisés UNIQUEMENT sur séances REC prévues (RPE max 3/10)
3. **INTERDICTION absolue** : Ajouts spontanés hors contexte prévu, même si "ça passe bien"

### Point Surveillance
Vérifier respect protocole sur 3 semaines consécutives S069-S071 avant envisager retour progressif terrain (actuellement indoor-only).

## 4. Dette Sommeil Facteur Limitant Validé

### Données S068
- **Moyenne durée** : 5.35h/nuit (-24% vs cible 7h)
- **VFC moyenne** : 40.5ms (faible, cible >50ms)
- **FC repos** : 47-53bpm (vs baseline 40bpm, +18-33%)
- **Scores Withings** : 47-70 (médians)

### Impact Performance
Malgré dette sommeil, séances Sweet-Spot exécutées avec qualité technique excellente grâce protocoles compensatoires :
- Sieste 20-30min pré-séance
- Hydratation renforcée
- Drills cooldown (moulinette/unijambiste)

### Limites Compensation
Protocoles efficaces **court terme uniquement**. VFC persistante faible (32-49ms) indique récupération parasympathique incomplète entre séances.

### Protocole Établi
- **VETO VO2 max** : Si sommeil <6h la veille, report séance ou substitution Z2
- **VETO Sweet-Spot haute intensité (>90% FTP)** : Si sommeil <6h + VFC <40ms
- **Objectif prioritaire S069-S071** : Progresser durée sommeil vers 6h+ systématique, cible 7h+ pour cycles intensifs futurs

## 5. Progression CTL Insuffisante pour Objectif FTP 260W

### Données S068
- **CTL début** : 57.9
- **CTL fin** : 58.2 (+0.3)
- **TSS hebdomadaire** : 371 (limite basse cible 380-420)

### Calculs Progression
Pour atteindre **FTP 260W** (+40W vs 220W actuel) en 20 semaines (estimation raisonnable âge 54) :
- **CTL cible** : ~75-80 (vs 58 actuel)
- **Progression requise** : +0.85-1.1 CTL/semaine
- **TSS hebdomadaire nécessaire** : 400-450 (vs 371 actuel)

### Facteurs Limitants Actuels
1. Dette sommeil chronique (5.35h/nuit)
2. Pattern surcharge non planifiée (compromis contrôle TSS)
3. Progression CTL stagnante (+0.3/semaine insuffisant)

### Protocole Ajustement S069
1. **TSS cible** : 400-420 hebdomadaire (+8-13% vs S068)
2. **Distribution** : Maintenir équilibre 70% endurance Z2 + 30% intensité Sweet-Spot/VO2
3. **Sommeil prioritaire** : 6h+ minimum pour absorption charge augmentée
