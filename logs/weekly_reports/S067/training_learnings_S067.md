# Enseignements d'Entraînement - Semaine S067

## Découvertes Techniques Majeures

### 1. VO2 Max : Capacité Actuelle 251W (114% FTP) Validée

**Contexte** : Premier test VO2 4×4min depuis plusieurs semaines, TSB -7 (non optimal).

**Découverte** :
- 4 intervalles @251W (114% FTP) complétés avec succès
- 5ème intervalle impossible à cette intensité = limite physiologique confirmée
- RPE 7/10 sur les blocs, effort contrôlé
- Récupération entre blocs adéquate (FC retour acceptable)

**Implications** :
- 251W = plafond actuel VO2 pour format 4×4min
- Progression possible par :
  - **Volume** : 4×4 → 5×4min à intensité réduite (245-250W)
  - **OU Intensité** : Maintien 4×4 à 255-260W (après progression FTP)
- Format 4×4 validé pour S068, pas d'augmentation prématurée

**Protocole associé** :
- Sommeil 9h déterminant pour exécution (malgré TSB -7)
- Hydratation renforcée (1×5min vs 1×10min) efficace
- Checklist VO2 violée (TSB +5 requis) mais compensée par sommeil

**À reproduire** :
- Hydratation 2 bidons isotonique + fréquence 1×5min
- Sommeil >9h si TSB <0
- Format 4×4 @251W = standard actuel

**À éviter** :
- Intensité >85% FTP à J-1 (violation extension NYC)
- Test VO2 si sommeil <7h (facteur limitant critique)

### 2. Sweet-Spot Adapté 77% : Alternative Validée TSB Négatif

**Contexte** : TSB -11 post-terrain, fatigue résiduelle, repos strict non souhaité.

**Découverte** :
- Format Sweet-Spot réduit à 77% FTP (vs 88-90% standard) parfaitement exécuté
- Précision remarquable : 197W réalisé vs 198W cible (<1W écart)
- Découplage 2.3% << 7.5% = qualité aérobie maintenue
- Cadence progression 90→92→92 RPM sans dégradation

**Implications** :
- **Zone "Sweet-Spot Léger"** validée : 75-80% FTP
- Utilisation TSB -10 à -5 quand repos strict non souhaité
- Maintien stimulus aérobie sans aggraver dette fatigue
- Alternative intelligente entre repos complet et Sweet-Spot standard

**Conditions d'application** :
- TSB entre -10 et -5
- Après surcharge récente (<48h)
- Sommeil qualité limitée mais >6h
- Aucune intensité >85% prévue dans 48h suivantes

**Format recommandé** :
- 3×8-10min @77-80% FTP
- Récupération 3-4min entre blocs
- Cadence libre optimale (88-92rpm)
- Durée totale 55-65min

**À intégrer** : Template "Sweet-Spot Recovery" pour situations similaires.

### 3. Extension Spontanée : Erreur de Perception Intensité

**Contexte** : Après Z2 75min correcte, extension 17min "sprints fun" nouveaux circuits NYC.

**Découverte CRITIQUE** :
- Perception : "Sprints légers, exploration ludique"
- Réalité mesurée : 98% FTP soutenu sur 17min (quasi-seuil prolongé)
- Charge métabolique : 28 TSS additionnel (+45% surcharge totale)
- Impact mesuré : Limitation VO2 lendemain (4 blocs vs potentiel 5+)

**Analyse Décalage Perception-Réalité** :

| Aspect | Perception Athlète | Réalité Métriques | Écart |
|--------|-------------------|-------------------|-------|
| Type effort | Sprints courts anaérobies | Quasi-seuil prolongé | +++++ |
| Intensité | "Fun, léger" | 98% FTP | +++++ |
| Charge TSS | ~10 TSS estimé | 28 TSS réel | +180% |
| Impact récup | "Aucun" | Sommeil 9h compensatoire | Majeur |
| RPE | 6-7/10 | 8/10 approprié | -1 |

**Facteurs Décalage** :
1. **Variété parcours** : Nouveauté NYC = engagement psychologique masquant intensité
2. **Absence référence** : Pas de comparaison mentale avec seuil standard
3. **Format irrégulier** : Sprints variés ≠ bloc soutenu homogène
4. **Motivation ludique** : Plaisir exploration diminue perception difficulté
5. **Fatigue résiduelle** : Déjà 75min Z2 = seuils perceptuels altérés

**Enseignement Fondamental** :
**Sprints VRAIS** (10-30sec, anaérobie, récup complète) ≠ **Efforts soutenus >90% FTP** (charge métabolique majeure)

| Type | Durée | Intensité | TSS/10min | Récup Requise |
|------|-------|-----------|-----------|---------------|
| Sprint vrai | 10-30sec | >120% FTP | ~5 TSS | Immédiate |
| Seuil court | 3-5min | 95-105% FTP | ~15 TSS | 48h |
| Quasi-seuil | 10-20min | 90-100% FTP | ~25 TSS | 72h |

Extension NYC = Quasi-seuil 17min, pas sprints.

**Protocole Correction** :

**Sessions "Fun" Autorisées** :
- Placement : J+2 post-repos OU J-3 minimum pré-intensité
- Format : Sprints <30sec VRAIS OU exploration Z2 stricte (<70% FTP)
- Durée extension : Maximum 15min
- Monitoring obligatoire : Vérification puissance post-séance

**Sessions "Fun" INTERDITES** :
- J-1 avant intensité planifiée (violation checklist)
- TSB <0 (dette fatigue)
- Sommeil <6h la veille
- Addition spontanée non planifiée

**Calibration Perception Future** :
- RPE seul insuffisant pour efforts irréguliers
- Vérifier données puissance PENDANT séance (Zwift companion)
- Si NP >85% FTP sur >5min = intensité élevée, pas "fun léger"
- Consulter TSS estimé avant décision extension

### 4. Discipline Terrain : Pattern Échec Confirmé

**Contexte** : Sortie outdoor 3h13, TSB -20 pré-séance (fatigue accumulée importante).

**Déclencheurs Surcharge Identifiés** :
1. **Présence autre cycliste** : Premières bosses effectuées à rythme poussé (compétitivité inconsciente)
2. **Relief/bosses** : Cadence basse 71rpm, intensité naturellement élevée
3. **Contrainte temporelle** : Jour tombant = pression finir avant nuit
4. **Erreur navigation** : Stress adaptation itinéraire en temps réel
5. **TSB négatif** : -20 = fatigue préexistante diminuant contrôle volontaire

**Résultat Mesuré** :
- IF 0.85 (85% FTP moyen sur 3h13) vs cible 0.65-0.75
- TSS 233 sur base fatiguée (TSB -20) = surcharge critique
- Puissance NP 187W vs cible ~145-165W pour endurance longue
- Découplage 3.4% correct mais masque intensité excessive

**Pattern Récurrent** :
```
Terrain + (Cycliste OU Bosses OU Contrainte) = Surcharge >+15% IF
```

Exemples historiques :
- S045 : IF 0.78 (cible 0.70), présence groupe
- S052 : IF 0.82 (cible 0.70), montées non planifiées
- S067 : IF 0.85 (cible 0.70), cycliste + bosses + timing

**Capacité Terrain vs Indoor** :
- Capacité objective terrain SUPÉRIEURE indoor (54km, relief, RPE 6/10)
- MAIS discipline intensité INFÉRIEURE (incapacité maintenir IF cible)
- Contraste total avec discipline indoor (précision <1W Sweet-Spot)

**Décision Stratégique Confirmée** :
**Indoor-only exclusif 2-3 mois** (jusqu'à janvier 2026 minimum)

**Justification** :
1. Développer capacité technique Sweet-Spot 90-92% sans risque surcharge
2. Consolider discipline intensité environnement contrôlé
3. Progresser FTP 220→235W sans interférences terrain
4. Valider auto-contrôle IF <0.75 sur séances longues indoor d'abord

**Conditions Réintroduction Terrain** :
- [ ] Sweet-Spot 92% validé 3×10min (S069-S070)
- [ ] FTP ≥230W confirmée (test S071)
- [ ] Discipline indoor parfaite 8 semaines consécutives
- [ ] Protocole terrain strict établi (solo, plat, IF monitoring temps réel)
- [ ] Test terrain contrôlé <2h, IF <0.75 validé 2× avant sorties libres

**Protocole Sortie Terrain Future** (quand autorisée) :
1. Solo obligatoire (0 autre cycliste)
2. Parcours plat/vallonné léger uniquement
3. Monitoring IF temps réel (Wahoo ELEMNT alerte si >0.75)
4. Pas de contrainte temporelle
5. Itinéraire backup téléchargé
6. TSB minimum +5
7. Durée progressive : 1h30 → 2h → 2h30 → 3h
8. Validation IF <0.75 sur 3 sorties avant liberté

### 5. Auto-Régulation Physiologique : Excellence Validée

**Observation** : Sommeil 9h nuit 14-15/11 (vs 5h33 habituel moyen) après surcharge jeudi.

**Découverte** :
- Organisme détecte automatiquement surcharge métabolique
- Compensation sommeil SANS décision consciente
- 9h récupération a permis exécution VO2 vendredi (malgré TSB -7)
- Pattern reproductible (observé semaines précédentes)

**Mécanisme Auto-Régulation** :
```
Surcharge détectée → Sommeil profond augmenté → Récupération accélérée
Extension NYC 28 TSS → 9h sommeil → VO2 4×4 possible lendemain
```

**Implications** :
1. **Signaux corporels fiables** : Sommeil >8h = indicateur surcharge récente
2. **Capacité récupération exceptionnelle** : FC repos 40bpm + auto-régulation
3. **Confiance physiologie** : Organisme compense erreurs tactiques
4. **MAIS limite** : Ne pas abuser - sollicitation répétée = surmenage

**Facteurs Auto-Régulation Efficace** :
- Âge 54 ans = expérience corporelle accrue
- FC repos 40bpm = fitness cardiovasculaire élevé
- Historique entraînement long (plusieurs années)
- Absence pathologies chroniques
- Nutrition/hydratation cohérentes

**À Surveiller** :
- Sommeil >9h répété = signal surcharge chronique (pas à reproduire)
- Auto-régulation ≠ autorisation surcharge volontaire
- Capacité récupération diminue avec accumulation fatigue
- Privilégier prévention vs correction

**Application S068** :
- Si sommeil spontané >8h = analyser surcharge veille
- Pas de test intensité si sommeil compensatoire détecté
- Respecter signaux repos même si TSB numérique OK
