## 2. GAPS CRITIQUES IDENTIFIÉS

- CTL optimal FTP 220W: **70.0**

**Impact Peaks Coaching**:
> "When you are 60 years young and your CTL drops from 80 down to 50, it's a long fight for months to get it back to 80!" (Hunter Allen)

À 54 ans, un CTL de 41.8 signifie:
- Fitness aérobie insuffisant pour supporter intensité FTP/VO2
- Risque élevé fatigue excessive si intensité prématurée
- Récupération ralentie (physiologie Masters 50+)

**Recommandation Peaks**:
- Phase: **RECONSTRUCTION BASE** (minimum 16 semaines)
- Progression: +2.5 CTL points/semaine
- Distribution: Tempo 35%, Sweet-Spot 20%, Endurance 25%
- Volume: 350 TSS/semaine (charge), 250 TSS (récup)
- Fréquence récup: Tous les **2 semaines** (Masters 50+)

**Gap Actuel**: Semaine S080 semble être test week (TSS bas), mais tests avec CTL 41.8 ne refléteront pas potentiel FTP réel une fois CTL reconstruit.

### 🟠 GAP #2: Timing Tests vs CTL Status

**Constat**:
- Tests programmés semaine S080 (Test FTP 20min, VO2 5min, Sprint 5sec, Anaérobie 1min)
- CTL actuel: 41.8 (critique)

**Principe Peaks Coaching**:
Tests de référence doivent être effectués quand:
- CTL ≥ 85% du CTL optimal pour FTP visé
- TSB entre -5 et +5 (forme neutre)
- Sommeil suffisant (7h+ nuit précédente)
- Pas d'intensité élevée <48h

**Gap Actuel**:
Tests effectués avec CTL sous-optimal peuvent:
- Sous-estimer FTP réel potentiel (manque endurance musculaire)
- Donner baseline basse qui sera dépassée rapidement après reconstruction CTL
- Risque: FTP mesuré ≠ FTP durable en conditions CTL optimal

**Recommandation**:
✅ **Accepter ces tests comme baseline "post-déclin CTL"**
- Établir FTP actuel dans état actuel
- Re-tester après 6-8 semaines reconstruction base (CTL ~55)
- Re-tester final après 16 semaines (CTL ~70) avant FTP target 260W

**Justification**:
Tests actuels = point de départ réaliste. Tests futurs mesureront progression CTL.

### 🟡 GAP #4: Distribution Intensité Post-Tests

**Constat Actuel (S080)**:
- 6 sessions tests: Sprint (AC), VO2, FTP, Anaérobie
- Distribution semaine tests = non représentative (par définition)

**Principe Peaks Coaching - Phase RECONSTRUCTION BASE**:
```
Recovery:      10% TSS
Endurance:     25% TSS ← FOCUS
Tempo:         35% TSS ← FOCUS PRINCIPAL
Sweet-Spot:    20% TSS ← FOCUS
FTP:            5% TSS
VO2:            3% TSS
AC/Neuro:       2% TSS
```

**Gap Anticipé (Risque Futur)**:
Après tests, tentation de:
- ❌ Augmenter FTP/VO2 trop tôt (CTL insuffisant)
- ❌ Faire trop de Z2 pure (inefficient budget temps 8-12h/semaine)
- ❌ Négliger Tempo/Sweet-Spot (zones les plus efficientes)

**Recommandation**:
Post-S080, semaines S081-S096 (16 semaines reconstruction):
- **Tempo (76-91% FTP)**: 35% du TSS total → ~122 TSS/semaine
  - Exemple: 3x 60min Tempo = 3x40 TSS = 120 TSS
- **Sweet-Spot (88-93% FTP)**: 20% → ~70 TSS/semaine
  - Exemple: 2x 45min Sweet-Spot = 2x35 TSS = 70 TSS
- **Endurance (56-75% FTP)**: 25% → ~87 TSS/semaine
  - Exemple: 1x 2h Z2 = 90 TSS

**Justification Hunter Allen**:
> "For Masters athletes, Tempo and Sweet-Spot are the biggest bang for your buck when time is limited. These zones build aerobic capacity efficiently without excessive fatigue."

## 6. ALERTES & RISQUES

### 🚨 Risque #1: Impatience Intensité
**Symptôme**: Envie augmenter FTP/VO2 trop tôt (avant CTL 55+)
**Impact**: Surentraînement, CTL stagne, fatigue chronique
**Prévention**: Validation distribution chaque semaine, alertes auto si FTP >5% ou VO2 >3%

### 🚨 Risque #2: Volume Insuffisant
**Symptôme**: Semaines charge <320 TSS (excuses: fatigue, météo, motivation)
**Impact**: CTL progression ralentie, objectif 16 semaines non atteint
**Prévention**: Compensation TSS proactive, indoor backup systématique

### 🚨 Risque #3: Récupération Sautée
**Symptôme**: 3+ semaines consécutives charge (Masters 50+ fatigue accumulation)
**Impact**: Burnout, maladie, blessure, drop CTL brutal
**Prévention**: Alerte automatique si pattern récup non respecté

### 🚨 Risque #4: Tests Prématurés
**Symptôme**: Tests FTP fréquents avant adaptation 6-8 semaines
**Impact**: FTP fluctuant, zones entraînement instables, démotivation
**Prévention**: Tests programmés S088 et S096 UNIQUEMENT (sauf indication médicale)
