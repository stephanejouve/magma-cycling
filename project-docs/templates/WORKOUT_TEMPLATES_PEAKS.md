**Contexte**: Phase RECONSTRUCTION_BASE (CTL 42.4 → 54-57)

## 📋 Vue d'ensemble

Templates optimisés pour phase **RECONSTRUCTION_BASE** selon méthodologie Peaks Coaching (Hunter Allen).

**Distribution cible**:
- **Tempo (35%)**: Zone principale reconstruction CTL
- **Sweet-Spot (20%)**: Efficacité maximale temps/bénéfice
- **Endurance (25%)**: Volume base aérobie
- **Recovery (10%)**: Récupération active
- **FTP/VO2/AC (10%)**: Intensité limitée (phase reconstruction)

**FTP baseline**: 223W (mesuré S080-02, 2026-02-10)

---

## 🎯 ZONE 1: TEMPO (76-91% FTP) - FOCUS 35%

### Rationale Peaks Coaching

> "Tempo is the bread-and-butter zone for Masters 50+ base building.
> It's hard enough to drive adaptation, gentle enough for high volume."
> — Hunter Allen, *Training & Racing with a Power Meter*

**Pourquoi Tempo en reconstruction CTL**:
- Sweet spot entre intensité et durabilité
- Développe capacité aérobie sans fatigue excessive
- Permet volume élevé (2-3 séances/semaine)
- Découplage cardio <5% si bien exécuté
- Compatible Masters 50+ (récupération 24-36h)

**Zones FTP 223W**:
- Z1 Tempo bas: **170-185W** (76-83% FTP)
- Z2 Tempo haut: **186-203W** (83-91% FTP)
- Cadence cible: **85-90rpm** (efficacité neuromusculaire)

---

### Template T1: Tempo Intervals (60min, 48-52 TSS)

**Objectif**: Construction base aérobie par intervalles Tempo

**Structure**:
```
Warmup (10min)
- 0-5min: 50-60% FTP (rampe progressive)
- 5-10min: 60-70% FTP

Main Set (40min)
- 3x (12min @ 85% FTP / 4min @ 60% FTP)
  → 12min @ 190W (85% de 223W)
  → 4min @ 134W (récupération active)
- Cadence: 85-88rpm (maintenir)
- Position: Guidon bas (aéro training)

Cooldown (10min)
- 0-5min: 65-55% FTP (descente progressive)
- 5-10min: 55-50% FTP
```

**Paramètres clés**:
- TSS estimé: **48-50**
- IF cible: **0.81-0.83**
- Découplage attendu: **<5%** (qualité)
- RPE: **5-6/10** (confortable mais soutenu)

**Hydratation**: 500ml/h (importante pour Tempo prolongé)

**Notes Masters 50+**:
- Si RPE >7/10 après intervalle 2: Réduire puissance 5W
- Si découplage >6%: Arrêter, passer en Endurance
- Récupération post: 24h minimum avant intensité

---

### Template T2: Tempo Endurance (60min, 45-50 TSS)

**Objectif**: Endurance tempo continue (volume + intensité modérée)

**Structure**:
```
Warmup (10min)
- 0-10min: Rampe 50-75% FTP

Main Set (40min)
- 40min @ 80-85% FTP continu
  → 179-190W (effort soutenu)
- Cadence: 85-88rpm (régulier)
- Pas de pause (effort continu)

Cooldown (10min)
- Rampe 70-50% FTP
```

**Paramètres clés**:
- TSS estimé: **46-48**
- IF cible: **0.80-0.82**
- Découplage attendu: **<6%**
- RPE: **5-6/10** (stable)

**Variations**:
- **V1 Indoor**: Tempo continu strict (power meter)
- **V2 Outdoor**: Tempo avec variabilité ±5% (terrain)

---

### Template T3: Tempo Long (90min, 70-75 TSS)

**Objectif**: Volume Tempo pour CTL progression

**Structure**:
```
Warmup (15min)
- Rampe 50-75% FTP

Main Set (60min)
- 60min @ 78-83% FTP
  → 174-185W (modéré)
- Split nutrition: Gel 30min, Gel 60min
- Hydratation: 600ml/h

Cooldown (15min)
- Rampe 70-50% FTP
```

**Paramètres clés**:
- TSS estimé: **72-75**
- IF cible: **0.78-0.81**
- Découplage attendu: **<7%** (acceptable pour 90min)
- RPE: **6-7/10** (modéré-dur)

**Masters 50+ Caution**:
- Maximum 1x/semaine (fatigue cumulative)
- Jour suivant: Recovery ou OFF
- Si découplage >8%: Stop, besoin récupération

## 🎯 ZONE 2: SWEET-SPOT (88-93% FTP) - FOCUS 20%

### Rationale Peaks Coaching

> "Sweet Spot is the 'Goldilocks zone' - not too hard, not too easy.
> Maximum CTL gain per hour invested."
> — Hunter Allen

**Pourquoi Sweet-Spot en reconstruction**:
- **Efficacité temps/bénéfice maximale** (TSS/h élevé)
- Juste sous seuil FTP (sustainable 20-30min)
- Améliore capacité lactate (économie)
- Moins taxant que FTP mais plus efficace qu'Endurance
- Idéal Masters 50+ (intensité contrôlée)

**Zones FTP 223W**:
- Z1 Sweet-Spot bas: **196-203W** (88-91% FTP)
- Z2 Sweet-Spot haut: **203-207W** (91-93% FTP)
- Cadence cible: **88-92rpm** (optimal)

---

### Template SS1: Sweet-Spot Intervals Classic (60min, 50-55 TSS)

**Objectif**: Intervalles Sweet-Spot standard (Peaks)

**Structure**:
```
Warmup (10min)
- 0-5min: 50-65% FTP
- 5-10min: 65-75% FTP (priming)

Main Set (40min)
- 3x (10min @ 90% FTP / 5min @ 60% FTP)
  → 10min @ 201W (Sweet-Spot)
  → 5min @ 134W (récupération active)
- Cadence: 88-90rpm (tenir constant)
- Respiration: Contrôlée (nasal possible)

Cooldown (10min)
- Rampe 70-50% FTP
```

**Paramètres clés**:
- TSS estimé: **52-55**
- IF cible: **0.84-0.86**
- Découplage attendu: **<5%** (excellent)
- RPE: **6-7/10** (dur mais contrôlable)

**Indicateurs qualité**:
- FC stable dans intervalles (ne monte pas +5 bpm)
- Cadence constante ±2rpm
- Respiration contrôlée (pas haletant)
- RPE stable (pas d'explosion intervalle 3)

---

### Template SS2: Sweet-Spot Extended (75min, 65-70 TSS)

**Objectif**: Volume Sweet-Spot pour CTL push

**Structure**:
```
Warmup (15min)
- Rampe 50-75% FTP progressive

Main Set (45min)
- 2x (15min @ 90% FTP / 7min @ 65% FTP)
  → 15min @ 201W (Sweet-Spot long)
  → 7min @ 145W (récupération)
- Cadence: 88-90rpm
- Nutrition: Gel à 40min

Cooldown (15min)
- Rampe 70-50% FTP
```

**Paramètres clés**:
- TSS estimé: **66-70**
- IF cible: **0.85-0.87**
- Découplage attendu: **<6%**
- RPE: **7/10** (limite haute)

**Masters 50+ Protocol**:
- Maximum 1x/semaine (fatigue neuronale)
- Si RPE >8/10 sur intervalle 2: Arrêter
- Récupération post: 48h minimum avant intensité

---

### Template SS3: Sweet-Spot + Tempo Combo (90min, 75-80 TSS)

**Objectif**: Combo efficacité (Sweet-Spot) + volume (Tempo)

**Structure**:
```
Warmup (15min)
- Rampe 50-75% FTP

Sweet-Spot Block (30min)
- 2x (10min @ 90% FTP / 5min @ 60% FTP)

Transition (5min)
- 5min @ 70% FTP

Tempo Block (25min)
- 25min @ 82% FTP (Tempo steady)
  → 183W

Cooldown (15min)
- Rampe 70-50% FTP
```

**Paramètres clés**:
- TSS estimé: **76-80**
- IF cible: **0.82-0.84**
- Découplage attendu: **<7%** (acceptable combo)
- RPE: **7-8/10** (workout difficile)

**Rationale**:
- Sweet-Spot crée stimulus qualité
- Tempo maintient TSS élevé sans surcharge
- Simule course réelle (variations intensité)

**Caution**: Réservé adherence >90% et CTL >45

## 📊 Utilisation Templates

### Sélection Template

**Phase RECONSTRUCTION_BASE (CTL <50)**:
- **Priorité 1**: Tempo Intervals (T1) - 2x/semaine
- **Priorité 2**: Sweet-Spot Classic (SS1) - 1x/semaine
- **Priorité 3**: Tempo Endurance (T2) - 1x/semaine
- **Éviter**: SS3 Combo (trop dur pour CTL bas)

**Phase CONSOLIDATION (CTL 50-70)**:
- Sweet-Spot Extended (SS2) - 1-2x/semaine
- Tempo Long (T3) - 1x/semaine
- Combo SS3 possible si adherence >90%

### Progression

**Semaines 1-2 (Ramp)**:
```
Lundi:    T1 (48 TSS)
Mercredi: SS1 (52 TSS)
Vendredi: T2 (46 TSS)
```

**Semaines 3-5 (Charge)**:
```
Lundi:    T1 ou T3 rotation
Mercredi: SS1 ou SS2 rotation
Vendredi: T2
```

**Semaine 6 (Récup + Test)**:
```
Lundi:    T1 (48 TSS)
Mercredi: OFF ou Recovery
Vendredi: Test FTP 20min
```

## 🎯 Success Metrics

**Qualité exécution**:
- ✅ Découplage <7% (Tempo/Sweet-Spot)
- ✅ IF ±0.02 de cible
- ✅ TSS ±5% de prévu
- ✅ RPE ressenti match zones

**Adherence**:
- ✅ >85% séances complétées
- ✅ Respect structure (pas de modification ad-hoc)
- ✅ Récupération respectée (jours OFF)

**Progression CTL**:
- ✅ +1.5-2.0 points/semaine (ramp)
- ✅ +2.5-3.0 points/semaine (charge)
- ✅ Pas de drop >1 point (sauf récup week)
