```
Nom : Sweet-Spot Recovery 77-80%
Durée : 55-65min

- Échauffement : 10min progressif 50→70% FTP

- Bloc 2 : 8-10min @77-80% FTP
- Récupération : 3-4min @50% FTP
- Bloc 3 : 8-10min @77-80% FTP
- Retour calme : 10min @50% FTP
```

**Caractéristiques** :
- Intensité : 77-80% FTP (vs 88-90% standard)
- TSS cible : 55-65 (modéré)
- Découplage attendu : <3%

**Conditions Application** :
- TSB entre -10 et -5
- Après surcharge récente (<72h)
- Sommeil >6h mais <7h30
- Pas d'intensité >85% FTP prévue dans 48h
- Alternative repos strict si motivation maintenue

**Validation** : S067-03 (TSS 58, IF 0.77, découplage 2.3%)

**Intégration** : Template créé pour S068+

### Critères Checklist VO2 Max (STRICTE)

**5 Critères OBLIGATOIRES** (aucune violation tolérée) :

1. **TSB ≥ +5** (strict)
   - Exception : TSB +3/+4 SI sommeil >9h ET aucun autre critère violé
   - Violation = report séance 24-48h

2. **Sommeil >7h la veille** (strict)
   - Cible optimale : >8h
   - <7h = report obligatoire
   - Qualité surveillée (sommeil profond >10%)

3. **Pas intensité >85% FTP dans 48h précédentes** (strict)
   - Inclut extensions spontanées, sprints longs (>1min), seuil
   - Violation = report séance 24h minimum

4. **FC repos matinale dans plage normale** (±5bpm vs baseline)
   - Baseline actuel : 40bpm
   - Alerte si >45bpm ou <35bpm
   - Variation >5bpm = analyser causes, possiblement reporter

5. **Aucun signe fatigue résiduelle** (subjectif mais critique)
   - Jambes lourdes au réveil = report
   - Motivation basse inhabituelle = vigilance
   - Alerte Riducks overtraining = report obligatoire

**Application S068** :
- Vérification systématique veille séance VO2
- Si 1 critère violé → report automatique, 0 négociation
- Sommeil >9h ≠ compensation autres violations (leçon S067)

## Surveillance Renforcée Identifiée

### Surveillance #1 : Pattern Sommeil Compensatoire

**Indicateur** : Sommeil >8h spontané après séance.

**Interprétation** :
- Signal surcharge métabolique détectée par organisme
- Auto-régulation efficace MAIS pas à reproduire volontairement
- Indique erreur planification OU exécution séance précédente

**Actions** :
1. Noter date + séance précédente + TSS
2. Analyser causes : Extension ? IF excessif ? TSB négatif ?
3. Repos strict OU récup active légère jour suivant
4. Pas de test intensité tant que sommeil >8h

**Seuil Alerte** : 2× sommeil >9h en 1 semaine = surcharge chronique → Semaine allégée obligatoire.

---

### Surveillance #2 : Découplage Tendance

**Méthode** : Moyenne mobile découplage Z2 (3 dernières séances).

**Seuils** :
- <2.5% : Excellent (forme optimale)
- 2.5-4% : Bon (nominal)
- 4-5.5% : Vigilance (fatigue légère)
- >5.5% : Alerte (surcharge, réduire charge)

**Fréquence** : Calcul hebdomadaire (dimanche soir).

**Application S068** :
- S067 moyenne : 1.9% (excellent)
- Maintien <3% objectif S068

---

### Surveillance #3 : Écarts RPE vs IF

**Méthode** : Noter écart pour chaque séance IF >0.75.

**Seuils** :
- Écart 0 à -0.5 : Calibration correcte
- Écart -1 : Acceptable (sous-estimation légère)
- Écart -1.5 : Vigilance (sous-estimation significative)
- Écart -2+ : Alerte (décalage perception majeur)

**Action Si Écart > -1.5** :
- Ignorer RPE complètement pour IF >0.80
- Monitoring puissance temps réel SEULE référence
- Vérification post-séance obligatoire

**S067** : Écart -1 systématique IF >0.80 → Surveillance maintenue S068.
